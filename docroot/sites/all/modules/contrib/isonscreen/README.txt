About
=====
Integrates the isOnScreen library into Drupal.



About isOnScreen
----------------

Library available at https://github.com/moagrius/isOnScreen

Simple jQuery plugin to determine if an element is within the viewport. Optional parameters allow the user to specify a minimum percentage of the element's dimensions that must be visible to qualify.

Installation
============

Dependencies
------------

- [Libraries API 2.x](http://drupal.org/project/libraries)
- [isOnScreen Library](https://github.com/moagrius/isOnScreen)

Tasks
-----

1. Download the isOnScreen library from https://github.com/moagrius/isOnScreen
2. Unzip the file and rename the folder to "isonscreen" (pay attention to the case of the letters)
3. Put the folder in a libraries directory
    - Ex: sites/all/libraries
4. The following files are required (last file is required for javascript debugging)
    - js/jquery.isonscreen.min.js
5. Ensure you have a valid path similar to this one for all files
    - Ex: sites/all/libraries/isonscreen/js/jquery.isonscreen.min.js

That's it!
